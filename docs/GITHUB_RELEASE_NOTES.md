# QuantumBot v2.0 Official - Release notes

QuantumBot v2.0 official è la base modulare stabile del progetto.

Questa release mantiene le funzionalità operative già testate e riorganizza il codice in un package Python più leggibile e manutenibile.

## Highlights

- Struttura modulare in `quantumbot/`
- Dashboard desktop Tkinter
- Engine separato
- Trading simulato
- CSV + SQLite
- Report e grafici PNG
- Supporto packaging macOS con PyInstaller

## Nota di sicurezza

Il progetto rimane in modalità simulazione. Non esegue ordini reali.
