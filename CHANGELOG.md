# Changelog

## v2.0 official

### Stato

Base modulare ufficiale di QuantumBot.

### Mantenuto

- Trading simulato
- Dashboard Tkinter
- Engine separato
- CSV `registro_trade.csv`
- SQLite `quantumbot.db`
- Report TXT/JSON/CSV
- Grafici PNG
- Auto Trade RSI
- Auto Profit/Loss
- Vendi EUR libero
- Vendi 25/50/75/100
- EngineGuard
- FeeAware
- MaxPositionsFix
- AutoCharts

### Struttura

Il codice è stato separato in package `quantumbot/`:

- `app.py`
- `dashboard.py`
- `database.py`
- `engine.py`
- `settings.py`
- `widgets.py`

### Note

La v1.0.19 originale stabile è conservata in `archive/` come riferimento storico.
